Crie um software que permita que o usuário entre com um nome de um pokemon e receba como resposta sua lista de habilidades.

Critérios:
* Utilizar orientação a objetos no seu software
* O resultado de cada consulta do usuário deve ser guardado em um arquivo que possua no nome o pokemon, a data e horário da tradução (ex: bulbasaur-10-10-18_10:30.txt)
Usar: https://pokeapi.co/


POKE API: 

    $ gem install bundler

    $ bundle install

    $ ruby poke.rb
